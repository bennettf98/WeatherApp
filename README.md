# ForecastFive

## About this Project

ForecastFive is a weather app built with HTML, CSS, and JavaScript allowing users to search for a specific location to view its 5-day weather forecast based on data provided by the Open-Meteo weather API. This project was created primarily to showcase my personal understanding of said languages. It uses Open-Meteo API for its forecast information.

## How it Works

Users of ForecastFive are presented with the ability to search for any city, state, or country. Upon doing so, they will be provided with a 5-day forecast for that location, including dates, precipitation (significance and % chance), sky conditions, and temperature highs and lows. Geolocation data is retrieved from Open-Meteo to pinpoint where user searched, then weather from subsequent location is retrieved and reflected into the forecast.

## Built With

* HTML
* CSS
* JavaScript
* Open-Meteo Geocoding API
* Open-Meteo Weather API
* Google Fonts
